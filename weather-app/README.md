# Weather App

This is a Python program that fetches real-time weather information for a city using an API.

## Features
- Takes city name as input
- Fetches live weather data from the internet
- Displays temperature and weather description

## Technologies Used
- Python
- Requests library
- API integration

## How to Run

1. Make sure Python is installed.
2. Install the required library:

pip install requests

3. Run the program:

python weather_app.py

## Example

Enter city name: Delhi

City: Delhi  
Temperature: 31 °C
