# Travel Planner CLI

This is a simple Python command-line application that suggests travel destinations based on the user's budget, trip duration, and preferred trip type.

## Features

- Takes user input from the terminal
- Suggests travel destinations based on budget
- Supports different trip types like mountains and beach
- Provides a basic itinerary suggestion based on number of days

## Technologies Used

- Python

## How to Run

1. Make sure Python is installed.
2. Clone this repository.
3. Navigate to the project folder.
4. Run the program:

python travel_planner_cli.py

## Example

Enter your budget: 20000  
Enter number of days: 4  
Trip type (mountains/beach): mountains  

Suggested destination: Manali  
Suggested itinerary: Explore local attractions and nature
